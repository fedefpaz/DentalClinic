Aclaraciones para un buen funcionamiento:

Se utilizó MySQL para el proyecto, por lo tanto se debe cambiar el username y la password. También está disponible y comentada la base de datos H2.

En caso de utilizar MySQL se debe crear la base de datos primero, recomiendo recargar la base cada vez que se hacen modificaciones para un funcionamiento más rápido.

Login ADMIN:
		username: fede@gmail.com
		password: password
Login USER:
		username: fabi@gmail.com
		password: password2

En Postman se debe enviar las autorizaciones guardadas en la carpeta login para obtener los privilegios de ADMIN y poder continuar.

En las vistas si se cuenta con los permisos de ADMIN se va a poder crear, ver y eliminar pacientes, dentistas y turnos.

Si se utilizan los permisos de USER solo se va a poder crear, ver y eliminar pacientes y turnos, restringiendo toda información sobre los dentistas.

Espero que puedan disfrutar al utlizar el proyecto tanto como lo disfruté cuando lo cree.
